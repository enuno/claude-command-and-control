# `openclaw logs`

Tail Gateway file logs over RPC (works in remote mode).

Related:

* Logging overview: [Logging](/logging)

## Examples

```bash  theme={null}
openclaw logs
openclaw logs --follow
openclaw logs --json
openclaw logs --limit 500
```
