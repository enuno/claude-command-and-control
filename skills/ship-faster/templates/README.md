# Templates

Templates are runnable, full projects you can start from. Each template is self-contained and includes its own entry documentation.

## Available templates

| Template | Description | Entry |
| --- | --- | --- |
| `001-copyback-studio` | CopyBack Studio full-stack app | [`001-copyback-studio/`](001-copyback-studio/) |
| `002-uniteconomics-console` | UnitEconomics Console analysis app | [`002-uniteconomics-console/`](002-uniteconomics-console/) |
| `003-marginledger` | MarginLedger unit-economics console for AI SaaS teams | [`003-marginledger/`](003-marginledger/) |
| `004-kanban-load-mirror` | Kanban Load Mirror workload analysis app | [`004-kanban-load-mirror/`](004-kanban-load-mirror/) |
| `005-multi-store-daily-brief` | Multi-Store Daily Brief ecommerce summary | [`005-multi-store-daily-brief/`](005-multi-store-daily-brief/) |
| `006-ticket-to-faq` | Ticket to FAQ knowledge base generator | [`006-ticket-to-faq/`](006-ticket-to-faq/) |
| `007-elevate-move-in-booking` | Elevate move-in booking system | [`007-elevate-move-in-booking/`](007-elevate-move-in-booking/) |

## Usage

1. Enter a template directory (example: `templates/001-copyback-studio`).
2. Follow the template README for install/build instructions.
