## üîë Key Metrics & Calculations

### Profitability Formula
**Source**: Economics PDF
```
Profit = (Bitcoin Mined √ó BTC Price) - (Energy Cost + Operating Costs)
```

### Hash Rate Calculation
**Source**: Handbook PDF
```
Total Hashrate = Number of Miners √ó Individual Miner Hashrate
```

### Energy Efficiency
**Source**: Heat Reuse PDF
```
Efficiency (J/TH) = Power Consumption (W) / Hashrate (TH/s)
```

### ROI Period
**Source**: Economics PDF
```
ROI Months = Initial Investment / Monthly Profit
```
