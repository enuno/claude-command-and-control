### Security Considerations

**Production Deployment**:
- Set `LANGFLOW_AUTO_LOGIN=False` (disable auto-login)
- Generate secure `LANGFLOW_SECRET_KEY` (32+ character random string)
- Configure specific `LANGFLOW_CORS_ORIGINS` (no wildcards)
- Deploy behind reverse proxy (Nginx/Caddy) with HTTPS
- Use managed PostgreSQL database (not SQLite)
- Enable API key authentication for all endpoints
- Rotate API keys every 90 days
- Implement rate limiting and monitoring

**Network Security**:
- Never expose port 7860 directly to internet
- Use Let's Encrypt for automatic HTTPS certificates
- Configure firewall rules and security groups
- Enable observability (LangSmith/LangFuse) for audit trails
