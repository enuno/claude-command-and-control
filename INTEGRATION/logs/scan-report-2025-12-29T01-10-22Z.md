**Web Scraping**:
- ‚ùå Fails on JavaScript-rendered sites (VuePress, VitePress, React)
- ‚ùå Limited to what's visible in static HTML
- ‚ö†Ô∏è No rate limiting but lower quality

- ‚úÖ Direct access to source markdown files
- ‚úÖ Repository metadata (stars, issues, languages)
- ‚úÖ File structure and navigation
- ‚ö†Ô∏è Requires authentication for large repos (rate limits)
- Use case: Open-source projects with documentation repos

**Best Practice**: For documentation sites with GitHub repositories, always prefer GitHub API extraction over web scraping.
