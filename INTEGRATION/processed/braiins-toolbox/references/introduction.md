# Introduction

The Braiins Toolbox is a comprehensive solution for efficiently managing your bitcoin mining operations. Toolbox offers you two powerful modes of operation: a user-friendly Graphical User Interface (GUI) and a flexible Command Line Interface (CLI). With these modes, you can choose the interface that best suits your preferences and workflow. Whether you are a beginner or an advanced user, Braiins Toolbox provides all the essential tools and features to streamline your batch management tasks and maximize your mining efficiency.

In this documentation, we will guide you through the various functionalities and capabilities of Braiins Toolbox, ensuring you have a clear understanding of how to utilize its full potential. It offers a range of features including:

- Ability to run batch operations on mining devices
- Network scanning to detect mining devices
- Remote installation of Braiins OS
- Configuration of various device settings
- Actions related to mining and the device system

## Prerequisites

- following architectures are supported:

Windows
macOS (running at least Monterey 12.4), both Intel and Apple silicon chips
Linux (64-bit), both x86_64 and aarch64
Linux (32-bit), armv7

- To be able to use Toolbox for the configurations and actions on mining devices, it is necessary the devices are running Braiins OS version 23.03 or newer. However, couple of features support also Antminers with Bitmain firmware and Whatsminers with MicroBT firmware.
- Each Braiins OS machine operates services on ports 22 (SSH port), 80 (HTTP port), and 50051 (public API port). To enable Toolbox recognition, ensure you can reach the machine via these ports, and configure your local network firewall to allow traffic through them.
- Users need to ensure that each device has access to the specified URL e5a33065.bos.braiins.com on port 3336. If this access cannot be established, the installation of Braiins OS will not proceed.
