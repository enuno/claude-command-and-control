# Claude Dispatch – Quickstart Checklist

## Prerequisites

- Claude Desktop app updated to the March 2026 release on macOS or Windows. [x](https://x.com/heynavtoor/status/2034679349157536233)
- Claude mobile app updated on iOS or Android. [x](https://x.com/heynavtoor/status/2034679349157536233)
- Active paid Claude plan (Max now; Pro rolling out). [x](https://x.com/heynavtoor/status/2034679349157536233)
- Stable internet connection on both desktop and phone. [x](https://x.com/heynavtoor/status/2034679349157536233)

***

## One-time Setup

1. **Enable Dispatch on Desktop** [x](https://x.com/heynavtoor/status/2034679349157536233)
   - Open Claude Desktop → go to **Cowork**.  
   - Click **Dispatch** in the left sidebar.  
   - Click **Get started**.  
   - Turn on:  
     - “Allow file access” (so Dispatch can use your Cowork setup).  
     - “Keep computer awake while Dispatch is active.”  

2. **Pair Your Phone** [x](https://x.com/heynavtoor/status/2034679349157536233)
   - On desktop, a QR code is displayed.  
   - On mobile, open the Claude app → tap **Dispatch**.  
   - Scan the QR code to link phone and desktop.  
   - Verify you see the same conversation on both devices.  

3. **Test Connectivity** [x](https://x.com/heynavtoor/status/2034679349157536233)
   - From your phone, send:  
     - `What files do you have access to right now?`  
   - Confirm Claude lists folders/files from your Cowork environment.  
   - If not, re-check Dispatch file-access toggle on desktop.

## Core Usage Patterns (Reliable Today)

Use short, focused commands from your phone while desktop stays awake. [x](https://x.com/heynavtoor/status/2034679349157536233)

- **Morning email triage**  
  - `Summarize my top 10 unread emails and flag anything urgent.` [x](https://x.com/heynavtoor/status/2034679349157536233)

- **File lookup**  
  - `Find the PDF I downloaded last week about the Q1 budget.` [x](https://x.com/heynavtoor/status/2034679349157536233)

- **Meeting prep**  
  - `Check my calendar for today and prepare briefing docs for each meeting using files from my project folder.` [x](https://x.com/heynavtoor/status/2034679349157536233)

- **Research summaries**  
  - `Read the three most recent documents in my Research folder and give me a one-paragraph summary of each.` [x](https://x.com/heynavtoor/status/2034679349157536233)

- **Notion / Slack catch-up**  
  - `Summarize my recent Notion notes.`  
  - `What were the key messages in the marketing Slack channel today?` [x](https://x.com/heynavtoor/status/2034679349157536233)

- **Expense processing**  
  - `Process all receipt images from my receipts folder and create a categorized spreadsheet.` [x](https://x.com/heynavtoor/status/2034679349157536233)

## Reliability Tips

- Keep tasks **single-purpose and information-focused** (summarize, list, retrieve, analyze). [x](https://x.com/heynavtoor/status/2034679349157536233)
- Define **skills** for repeated workflows (e.g., meeting prep, expenses) and reference them with terse mobile prompts. [x](https://x.com/heynavtoor/status/2034679349157536233)
- Add to Global Instructions:  
  - `When receiving a message via Dispatch, confirm the task before executing and ask for clarification if any step is ambiguous.` [x](https://x.com/heynavtoor/status/2034679349157536233)
- Test each new workflow from your phone while watching the desktop until it succeeds 3× in a row. [x](https://x.com/heynavtoor/status/2034679349157536233)

## Daily 5-Minute Routine (Template)

- **Wake-up:**  
  - `Good morning. Summarize my top emails, check my calendar for today, and list any files that changed in my project folder yesterday.` [x](https://x.com/heynavtoor/status/2034679349157536233)
- **Breakfast:**  
  - Review briefing, then:  
  - `Draft a reply to the email from [person] about [topic]. Keep it under 100 words.` [x](https://x.com/heynavtoor/status/2034679349157536233)
- **Commute:**  
  - `Prepare a briefing doc for my 10am meeting using the latest files in my project folder.` [x](https://x.com/heynavtoor/status/2034679349157536233)
